# HTML Basics - README

## Introduction

HTML (**HyperText Markup Language**) est le langage utilisé pour créer la structure des pages web.
Il permet d’organiser le contenu d’un site internet à l’aide de **balises (tags)**.

---

# 1. Structure de base d'une page HTML

Une page HTML5 commence toujours par une structure de base appelée **squelette HTML**.

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>My first page</title>
</head>
<body>

</body>
</html>
```

### Explication

* `<!DOCTYPE html>` : indique que le document utilise **HTML5**
* `<html>` : balise racine de la page
* `<head>` : contient les **métadonnées**
* `<body>` : contient le **contenu visible**

---

# 2. Le contenu du `<head>`

La balise `<head>` contient des informations importantes sur la page :

* **metadata**
* **title de la page**
* **liens vers**
